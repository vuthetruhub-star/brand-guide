## Visual Identity

### Style Direction
- **Aesthetic:** High-contrast, authentically retro
- **Mood:** Dark and edgy, terminal/hacker
- **Era/Inspiration:** 90s hacker culture, CRT monitors, early terminal interfaces

### Color Palette
| Role            | Color Name    | Hex       | Usage                          |
|-----------------|--------------|-----------|--------------------------------|
| Primary         | Terminal Green| #00FF41   | CTAs, highlights, active text  |
| Secondary       | Dim Green    | #0D7A2E   | Secondary elements             |
| Accent          | Bright Green | #33FF66   | Hover states, focus rings      |
| Background      | Black        | #000000   | Page background                |
| Surface         | Dark Grey    | #0A0A0A   | Cards, panels                  |
| Text Primary    | Green        | #00FF41   | Headings, main text            |
| Text Secondary  | Dim Green    | #00CC33   | Descriptions                   |
| Border          | Dark Green   | #003300   | Dividers, outlines             |

### Typography
| Role          | Font Family | Weight | Size  |
|---------------|------------|--------|-------|
| Display/Hero  | Termina    | 700    | 64px  |
| Heading H1    | Termina    | 700    | 48px  |
| Heading H2    | Termina    | 600    | 32px  |
| Body          | Termina    | 400    | 16px  |
| Code/Mono     | Termina    | 400    | 14px  |
| Button/Label  | Termina    | 600    | 14px  |

### Effects & Textures
- **Shadows:** None or very subtle green glow
- **Gradients:** No
- **Noise/Grain:** CRT scanline effect, subtle noise overlay
- **Animations:** Terminal typing effect, cursor blink, glitch on hover
- **Hover effects:** Green glow, text flicker

### Do's
- Always use monospace/Termina font
- Keep everything high-contrast (green on black)
- Use CRT/terminal visual effects
- Keep borders sharp (0px radius)
- Use uppercase for labels and buttons

### Don'ts
- Don't use rounded corners
- Don't use pastel or muted colors
- Don't use serif or decorative fonts
- Don't use gradients or soft shadows
- Don't use stock photography
